## ‚ö° The Fastest Path to Success

### 10-Minute Quick Start

**Step 1: Clone the repository**
```bash
git clone https://github.com/MUSTAQ-AHAMMAD/store-opening-ai.git
cd store-opening-ai
```

**Step 2: Run setup (automatic)**
```bash
./setup.sh        # Mac/Linux
setup.bat         # Windows
```

**Step 3: Start the application**
```bash
# Terminal 1
./start_backend.sh

# Terminal 2
./start_dashboard.sh
```

**Step 4: Open browser**
- Go to: http://localhost:8501
- Login: admin / admin123

**Step 5: Test AI**
- Click "Tasks & Checklists"
- Click "Send Follow-up" on any task
- Check Terminal 1 for AI output!

**‚úÖ Done! You're now running the system!**

For details, see **[STEP_BY_STEP.md](./STEP_BY_STEP.md)**
