## üìû Voice Calling System

### Escalation Levels:
- **Level 0** (0-2 days overdue): WhatsApp with AI message
- **Level 1** (3-6 days overdue): Voice call to team member
- **Level 2** (7+ days overdue): Voice call to manager

### Call Features:
- **Automated calling** via Twilio
- **AI-generated scripts** - contextual messages
- **Interactive menus** - press 1 to acknowledge
- **Call logging** - track all interactions
- **Fallback SMS** - if call fails
- **Manager escalations** - automatic hierarchy

### Call Flow:
1. System detects overdue task
2. Determines escalation level
3. Generates AI message
4. Initiates Twilio call
5. Plays script to recipient
6. Captures acknowledgment
7. Logs result
8. Triggers follow-up actions
